# 코딩 연습 - 예외 처리하기

`Program.cs`에 아래 과제들의 코드를 순서대로 작성하세요.
모든 과제를 작성한 후 실행하면 각 과제의 실행 결과가 순서대로 출력됩니다.

> 💡 변수 이름이 겹치는 경우 중괄호 `{ }`로 코드를 감싸면 각 블록에서 같은 이름을 사용할 수 있습니다.

---

## 1. 기본 try-catch

배열 범위를 벗어나는 접근에서 예외를 처리하세요.

**실행 결과**

```
에러가 발생했습니다.
```

---

## 2. Exception 변수 사용

catch 블록에서 예외 정보를 받아 상세한 내용을 출력하세요.

**실행 결과**

```
예외 발생: Index was outside the bounds of the array.
```

---

## 3. 특정 예외 처리

`FormatException`을 지정하여 형식 변환 오류를 처리하세요.

**실행 결과**

```
에러 발생: The input string '3.14' was not in a correct format.
3.14는 정수여야 합니다.
```

---

## 4. 변수 없이 catch 사용

예외 정보가 필요 없으면 변수를 생략하여 
`DivideByZeroException`을 처리하세요.

**실행 결과**

```
0으로 나눌 수 없습니다.
```

---

## 5. 예외 필터 (when)

`when` 절을 사용하여 조건부로 예외를 처리하세요.

**실행 결과**

```
404 오류: 페이지를 찾을 수 없습니다.
```

---

## 6. finally 사용

try-catch-finally 문에서 finally 블록이 예외 발생 시에도 실행되는지 확인하세요.

**실행 결과**

```
예외 발생: Attempted to divide by zero.
프로그램을 종료합니다.
```

---

## 7. 예외가 없을 때도 finally 실행

예외가 발생하지 않는 경우에도 finally 블록이 실행되는지 확인하세요.

**실행 결과**

```
10 / 2 = 5
계산이 완료되었습니다.
```


## 11. throw 문

`throw` 문을 사용하여 0으로 나누기 시도 시 예외를 발생시키는 
`Divide` 메서드를 작성하세요.

**실행 결과**

**실행 결과**

## 13. 예외 다시 던지기
